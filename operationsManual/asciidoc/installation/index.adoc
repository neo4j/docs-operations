= Installation

Neo4j can be installed in different deployment contexts, such as Linux, macOS, and Windows.

The following topics are covered:

* <<deployment-requirements, System requirements>> -- The system requirements for a production deployment of Neo4j.
* <<neo4j-browser, Neo4j Browser>> -- About Neo4j Browser.
* <<neo4j-desktop, Neo4j Desktop>> -- About Neo4j Desktop.
* <<linux-installation, Linux>> -- Installation instructions for Linux.
* <<osx-installation, macOS>> -- Installation instructions for macOS.
* <<windows-installation, Windows>> -- Installation instructions for Windows.

.Installation-free options
[TIP]
====
*Neo4j AuraDB* is a fully managed Neo4j database, hosted in the cloud and requires no installation.
For more information, see the link:https://neo4j.com/aura/[AuraDB product page] and link:https://neo4j.com/docs/aura/current/[AuraDB documentation].
