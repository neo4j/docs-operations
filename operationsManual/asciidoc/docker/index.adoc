= Docker

Neo4j can be run in a Docker container.

This chapter describes the following:

* <<docker-overview, Introduction>> -- Introduction to running Neo4j in a Docker container.
* <<docker-neo4j-configuration, Configuration>> -- How to configure Neo4j to run in a Docker container.
* <<docker-cc, Clustering>> -- How to set up Causal Clustering when using Docker.
* <<docker-operations, Docker specific operations>> - Descriptions of various operations that are specific to using Docker.
* <<docker-security, Security>> - Information about using encryption with the Docker image.
* <<docker-maintenance, Docker maintenance operations>> How to maintain Neo4j when running in a Docker container.
* <<docker-ref-settings, Docker specific configuration settings>> - A conversion table for the Neo4j configuration settings to Docker format.

[NOTE]
Docker does not run natively on macOS or Windows.
For running Docker on macOS and Windows, please consult the https://docs.docker.com/engine/installation[documentation provided by Docker].
