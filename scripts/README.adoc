compare-ref-settings.py extracts the Neo4j keys from the "Neo4j format" column of the Docker ref table
in modules/ROOT/pages/docker/ref-settings.adoc, extracts keys from 3rd-level headings (=== `key`) in
modules/ROOT/pages/configuration/configuration-settings.adoc and prints the differences (only-in-ref, only-in-config).

To run the script, use the following command in your terminal:

[source, python]
----
python3 scripts/compare-ref-settings.py \
    --ref modules/ROOT/pages/docker/ref-settings.adoc \
    --config modules/ROOT/pages/configuration/configuration-settings.adoc
----
