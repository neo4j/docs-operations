= Property-based access control (PBAC)

Property-based access control works as on a standard database.
For details, see xref:authentication-authorization/property-based-access-control.adoc[Property-based access control].
In 2026.07, PBAC only supports `READ` privileges.

[NOTE]
====
Privileges are granted on the *virtual database*.
Explicitly granting privileges on a shard is not supported.
====

The following example shows how to grant permission to `READ` the `address` property on `Email` or `Website` nodes with domain `exampledomain.com` to role `regularUsers`:

[source, cypher]
----
GRANT READ { address } ON GRAPH `foo-sharded` FOR (n:Email|Website) WHERE n.domain = 'exampledomain.com' TO regularUsers;
----
