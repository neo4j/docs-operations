[[validate-backup]]
= Validate a database backup
:description: This section describes how to validate a collection of backups.

[[validate-backup-command]]
== Command

The `neo4j-admin backup validate` command validates a collection of backups created using the `neo4j-admin backup` command.
It is available only for sharded-property databases.

[role=label--changed-2025.12]
[[validate-backup-syntax]]
=== Syntax

[source,role=noheader]
----
neo4j-admin backup validate [-h] [--expand-commands] [--verbose]
                            [--additional-config=<file>] --database=<database>
                            [--format=<value>] --from-path=<backup-path>
----


=== Description

Command to validate a collection of backups.

[[validate-backup-command-options]]
