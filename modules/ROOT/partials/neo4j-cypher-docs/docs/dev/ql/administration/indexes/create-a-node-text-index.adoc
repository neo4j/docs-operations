== Create a node text index ==
A named text index on a single property for all nodes with a particular label can be created with `CREATE TEXT INDEX index_name FOR (n:Label) ON (n.property)`. Note that the index is not immediately available, but is created in the background.

include::includes/administration-indexes-create-a-node-text-index.query.adoc[]


Note that text indexes only recognize string values, do not support multiple properties, and that the index name must be unique.
