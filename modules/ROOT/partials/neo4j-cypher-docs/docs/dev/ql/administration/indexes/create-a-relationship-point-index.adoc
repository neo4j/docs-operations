== Create a relationship point index ==
A named point index on a single property for all relationships with a particular relationship type can be created with `CREATE POINT INDEX index_name FOR ()-[r:TYPE]-() ON (r.property)`. Note that the index is not immediately available, but is created in the background.

include::includes/administration-indexes-create-a-relationship-point-index.query.adoc[]


Note that point indexes only recognize point values, do not support multiple properties, and that the index name must be unique.
