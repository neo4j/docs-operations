== Create a relationship text index ==
A named text index on a single property for all relationships with a particular relationship type can be created with `CREATE TEXT INDEX index_name FOR ()-[r:TYPE]-() ON (r.property)`. Note that the index is not immediately available, but is created in the background.

include::includes/administration-indexes-create-a-relationship-text-index.query.adoc[]


Note that text indexes only recognize string values, do not support multiple properties, and that the index name must be unique.
