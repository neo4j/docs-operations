== Create a single-property b-tree index for relationships ==
A named b-tree index on a single property for all relationships with a particular relationship type can be created with `CREATE INDEX index_name FOR ()-[r:TYPE]-() ON (r.property)`. Note that the index is not immediately available, but is created in the background.
