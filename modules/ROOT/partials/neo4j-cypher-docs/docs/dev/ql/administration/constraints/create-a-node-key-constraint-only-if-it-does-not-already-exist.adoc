== Create a node key constraint only if it does not already exist ==
If it is not known whether a constraint exists or not, add `IF NOT EXISTS` to ensure it does. The node key constraint ensures that all nodes with a particular label have a set of defined properties whose combined value is unique and all properties in the set are present.
