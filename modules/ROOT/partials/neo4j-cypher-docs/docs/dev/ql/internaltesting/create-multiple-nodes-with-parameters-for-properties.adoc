[[internaltesting-create-multiple-nodes-with-parameters-for-properties]]
== Create multiple nodes with parameters for properties ==

Use `UNWIND` to create multiple nodes from a parameter.



.Parameters
[source,javascript]
----
{
  "props" : [ {
    "name" : "Andy",
    "position" : "Developer"
  }, {
    "name" : "Michael",
    "position" : "Developer"
  } ]
}
----
